# aquaterm

aquazone 風のターミナル熱帯魚育成ツール。Rust + crossterm による自前ハーフブロック描画で、ブルー系の水槽に熱帯魚が泳ぎます。餌をやって育て、成長・繁殖させて眺めるアクアリウムです。

## 特徴

- crossterm による自前レンダリング(ratatui 非依存)。ハーフブロック文字 `▀` で縦2倍解像度のカラー描画、前フレームとの差分のみ書き込みちらつきを抑制
- 魚は f64 座標で保持しなめらかに遊泳(ランダムウォーク+慣性、壁で緩やかに反射、同種で軽く群れる、空腹だと餌に寄る)
- 3種類の魚(ネオン=青系 / 金魚=オレンジ / グッピー=白+差し色)。稚魚と成魚で見た目が変わる
- 育成要素: 空腹度(見た目に反映)・成長(稚魚→成魚)・産卵→孵化・病気と治療・死亡
- 一時停止・速度変更・グレートリセットで観察やデバッグがしやすい
- 終了時に状態を保存し、次回起動で続きから再開

## ビルドと起動

```sh
cargo build --release
./target/release/aquaterm
```

初回起動時はヘルプが自動表示されます(何かキーを押すと水槽へ)。

## キー操作

| キー      | 動作                                         |
|-----------|----------------------------------------------|
| `f`       | 餌やり(中央上部から3〜5粒投下)              |
| `m`       | 薬を投げる(病気の魚が触れると治癒)          |
| `p`       | 一時停止 / 再開                              |
| `[` / `]` | シミュレーション速度を下げる / 上げる         |
| `R`       | 水槽グレートリセット(確認あり)              |
| `+` / `-` | 魚を1匹追加 / 間引く(デバッグ用)            |
| `,`       | 設定画面(将来対応予定)                      |
| `?`       | ヘルプの開閉                                 |
| `q` / Esc | 終了(状態を保存)                           |

## 育成のしくみ

- **空腹度**: 時間で減少し、餌を食べると回復する。空腹になると最寄りの餌へ寄ってくる。満腹/普通/腹ぺこの3段階で泳ぐ速さと色が変わる(腹ぺこは速く泳ぎ色が薄暗くなる)
- **成長**: 満腹を維持し続けると稚魚が成魚に育ち、サイズが大きくなる
- **繁殖(産卵→孵化)**: 満腹の成魚は水底付近に卵を産む。卵は一定時間で孵化して稚魚になる(個体数上限を超える分は孵化しない)
- **病気**: 腹ぺこが続いたり過密になると発症する。まだらにくすんだ色になり泳ぎが鈍り、成長・繁殖が止まる。放置すると弱って力尽きる。`m` の薬で治療できる(健康な魚には無害)
- **死亡**: 空腹度0や病気の放置が続くと力尽きて水槽から消える。状態はステータスバーに表示される
- **時間制御**: `p` で全体を一時停止、`[` `]` で速度を段階変更(空腹度減少・成長・孵化など時間経過ロジック全体に倍率がかかる)

## 状態の永続化

`~/.config/aquaterm/state.json` に各魚の種類・成長段階・空腹度・病気状態・座標、卵・餌・薬の状態、経過時間を保存します。`q` 終了時に保存し、起動時に読み込んで再開します(ファイルが無ければ初期状態で開始)。

## テスト

```sh
cargo test
```

空腹度の減衰・成長判定・繁殖(個体数上限を含む)・餓死などの育成ロジックを単体テストで検証しています。
