aquazone 風のターミナル熱帯魚育成ツール。Rust + crossterm による自前ハーフブロック描画で、ブルー系の水槽に熱帯魚が泳ぎます。餌をやって育て、成長・繁殖させて眺めるアクアリウムです。

![screenshot](docs/screenshot.png)

## 特徴

- crossterm による自前レンダリング(ratatui 非依存)。ハーフブロック文字 `▀` で縦2倍解像度のカラー描画、前フレームとの差分のみ書き込みちらつきを抑制
- 魚は f64 座標で保持しなめらかに遊泳(ランダムウォーク+慣性、壁で緩やかに反射、同種で軽く群れる、腹ぺこだと通常の遊泳よりはっきり優先して餌に一直線に向かう)。低頻度・ランダムなタイミングで一瞬だけ速く動く「瞬発ダッシュ」も入る
- 4種類の魚(ネオン=青系 / 金魚=オレンジ / グッピー=白+差し色 / サメ=灰青の大型種)。稚魚と成魚で見た目が変わり、サメも含めて空腹度・成長・病気・死亡・寿命の育成ロジックには参加する
- サメは捕食者: 空腹時に他の魚を検知して追跡(追跡中は通常種より速い)して襲い、捕食されると強化した血飛沫演出(複数色の破片パーティクル+同心円状に広がる血の滲み)とともにその場で消える(頻度は控えめ・クールダウンあり)。捕食するたびに段階的に大きくなる。**サメの入手経路は`S`キーのみ**(初期配置・`+`キーのランダム追加・産卵→孵化のいずれにもサメは含まれない)
- 通常の魚は、サメの空腹状態に関わらず近くにサメがいると常に回り込むように(ジグザグに)逃げる。ガラスを叩かれた驚き逃げも同様に機敏でジグザグ、いずれも逃走のたびに空腹度を消費する
- 育成要素: 空腹度(見た目・オーバーレイに反映)・成長(稚魚→成魚、その後も満腹維持でさらに段階的にサイズアップ)・産卵→孵化(サメを除く3種のみ)・老齢での確定産卵と寿命死(全種共通)・病気と治療・死亡(猶予は長め、死亡演出つき)
- カーソル(照準)で餌・薬の投下位置を指定でき、投下位置には一瞬光る演出が出る。ガラスを叩いて(`t`)近くの魚を驚かせることもできる(叩きすぎるとストレスで病気になりやすくなる)
- 自動モード(`a`): ONにすると、腹ペコ/病気の魚がいるときに自動で餌やり・投薬を行い、低頻度でランダムにガラスも叩く(いずれも一定間隔のクールダウンあり)
- 設定画面(`,`): 効果音・オーバーレイ・自動モードのON/OFFを一覧して切り替えられる
- 各魚の直上にステータスオーバーレイ(腹ペコ/病気フラグ・生命残りゲージ)を点滅表示(ON/OFF切替可)
- カニは観賞用の背景生物として水底を歩く(育成ロジック対象外)。水底に沈んだ食べ残しの餌・薬を片付ける掃除役も兼ねる(1tickで1つまでのゆっくりした掃除)
- 個体数上限は端末サイズに応じて最大100匹まで。`+`キーでの直接追加は50匹まで、それ以上は産卵→孵化でのみ到達する
- 効果音(周波数スイープ+速い減衰で合成したビープ音、ON/OFF切替可)。オーディオデバイスが無い環境でもクラッシュしない
- 起動時にタイトルロゴのスプラッシュ画面を表示(`?` のヘルプ画面にも同じロゴ)。ヘルプ画面には通常3種+サメの図鑑(実際のドット絵スプライト付き)も表示する(端末が狭い場合はテキストのみにフォールバック)
- 一時停止・速度変更・グレートリセットで観察やデバッグがしやすい
- 終了時に状態を保存し、次回起動で続きから再開

## ビルドと起動

```sh
cargo build --release
./target/release/aquaterm
```

起動時に毎回タイトルロゴのスプラッシュ画面が出ます(何かキーを押すか少し待つと消えます)。初回起動時はさらにヘルプが自動表示されます(何かキーを押すと水槽へ)。

## キー操作

| キー      | 動作                                                   |
|-----------|--------------------------------------------------------|
| 矢印キー  | カーソル(照準)を移動                                  |
| `f`       | カーソル位置から餌やり(3〜5粒投下、投下位置が一瞬光る) |
| `m`       | カーソル位置から薬を投げる(病気の魚が触れると治癒)     |
| `t`       | ガラスを叩く(近くの魚が驚いて逃げる。叩きすぎ注意)     |
| `p`       | 一時停止 / 再開                                        |
| `[` / `]` | シミュレーション速度を下げる / 上げる                   |
| `R`       | 水槽グレートリセット(確認あり)                        |
| `v`       | ステータスオーバーレイ(腹ペコ/病気/生命ゲージ)の表示切替 |
| `s`       | 効果音(SE)のON/OFF                                    |
| `a`       | 自動モード(自動餌やり/投薬/ガラス叩き)のON/OFF        |
| `+` / `-` | 魚を1匹追加(通常3種からランダム) / 間引く(デバッグ用) |
| `S`       | サメを1匹確実に追加(サメを増やせる唯一の方法)         |
| `,`       | 設定画面を開く(効果音/オーバーレイ/自動モードを一覧切替) |
| `?`       | ヘルプの開閉(通常3種+サメの図鑑つき)                  |
| `q` / Esc | 終了(状態を保存)                                     |

設定画面(`,`)を開いている間は、上下キーで項目選択・Enter/スペースでトグル・Escで画面を閉じます(水槽の操作には伝わりません)。

## 育成のしくみ

- **空腹度**: 時間で減少し、餌を食べると回復する。腹ぺこになると通常の遊泳(ランダムウォーク・群れ)よりはっきり優先して最寄りの餌へ一直線に向かう。満腹/普通/腹ぺこの3段階で泳ぐ速さと色も変わる(腹ぺこは速く泳ぎ色が薄暗くなる)
- **ステータスオーバーレイ**: 各魚の直上に、腹ペコ/病気フラグと「生命残りゲージ」(空腹度と病気の深刻度から算出、緑〜黄〜赤のグラデーション)を点滅表示する。`v` キーでON/OFFを切り替えられる
- **成長**: 満腹を維持し続けると稚魚が成魚に育ち、サイズが大きくなる。成魚になった後も満腹維持を続けると、さらに段階的にサイズアップする(全種共通・上限あり。大きくなるほど泳ぐ速度がわずかに遅くなる)
- **繁殖(産卵→孵化)**: 満腹の成魚は水底付近に卵を産む(頻度は低め・数十分に1回程度を目安に調整済み)。卵は一定時間で孵化して稚魚になる(個体数上限を超える分は孵化しない)。**サメは産卵しない**(サメが増える唯一の方法は`S`キー)
- **寿命・世代交代**: 各個体は生まれてからの経過時間(年齢)を記録し、老齢に達すると満腹状態などの条件を問わず確定で1回だけ産卵する(次世代を残す最後のチャンス。サメは対象外)。さらに年を取ると老衰で寿命死する(既存の死亡演出を使う)。対象は全種共通(サメも含む。サメが無限に居座り続けない)
- **病気**: 腹ぺこが続いたり過密になると発症する(頻度は低め・最高速度で遊んでいて水槽全体で10分に1回程度を目安に調整済み)。まだらにくすんだ色になり泳ぎが鈍り、成長・繁殖が止まる。`m` の薬で治療できる(健康な魚には無害)。ガラスを叩きすぎると周辺の魚が一時的に発症しやすくなる
- **死亡**: 空腹度0または病気の放置が長時間(それぞれ猶予10分以上)続くと力尽きる。死んだ魚は仰向けのスプライトに変わってゆっくり水面近くまで浮上し、しばらく(15秒程度)水槽に留まったのち消える。餌や薬を与えて回復すれば死亡は避けられる
- **サメの捕食**: サメは空腹時、他の魚(サメ以外)を検知して追跡し、触れると捕食する。追跡(捕食モード)中だけサメの速度が通常3種の最高速度よりはっきり速くなる(巡回中は特別早くしない)。捕食された魚は強化した血飛沫演出(内臓を思わせる複数色の破片パーティクル+同心円状に広がって数秒間まっかに見えてからフェードアウトする血の滲み)とともに即座に消え、サメの空腹度は大きく回復し、サメ自身は捕食するたびに段階的に大きくなる(上限あり)。連続捕食を防ぐクールダウンがあり、頻度は控えめ
- **サメへの警戒・逃走**: 通常の魚は、サメが空腹かどうかに関わらず近くにサメがいると常に逃げる(「みんなサメが嫌い」)。逃走は真っ直ぐ離れるだけでなく回り込むようにジグザグに切り返し、通常の遊泳よりはっきり優先される機敏な動きになる。ガラスを叩かれた驚き逃げも同様にジグザグかつ機敏。いずれの逃走も、始まった瞬間に空腹度を消費する(逃げるのにエネルギーを使う想定。危険が続く間の再課金はない)
- **ランダムな瞬発ダッシュ**: サメ・餌などのトリガーが無い通常の遊泳中でも、低頻度・ランダムなタイミングで一瞬だけ速く動く(数十秒に1回あるかないか程度)
- **サメの入手経路**: `S`キーのみ。初期配置・グレートリセット・`+`キーのランダム追加・産卵→孵化には一切含まれない(狙って入れたときだけ水槽に現れる)
- **カニ**: 水底を左右に歩き、時々立ち止まる観賞用の生物(育成ロジック対象外)。水底に停留した食べ残しの餌・薬に近づくと片付ける掃除役も兼ねる(1匹のカニが1tickで片付けられる餌・薬はそれぞれ1つまでなので、山になっていても一気には無くならずゆっくり片付く)
- **水底の食べ残し**: 餌・薬は水底に着くと停留し(寿命が止まる)、カニに片付けられるか、停留数が上限を超えると古いものから消える。1tickで1匹の魚が消費できる餌・薬は1つまでなので、密集させても一気には無くならない
- **個体数上限**: 端末サイズ(論理ピクセル数)に応じて決まる。`+`/`S`キーでの直接追加は50匹まで。それ以上(50〜端末依存の上限・最大100)は産卵→孵化を経由してのみ到達する(ただしサメは産卵しないため、サメの数は`S`キーを押した回数分までにしかならない)
- **自動モード(`a`)**: ONの間、腹ペコの魚がいて漂っている餌が少なければ一定間隔(数十秒に1回程度)で自動的に餌やりし、病気の魚がいて漂っている薬が少なければ同様に自動投薬する。加えて低頻度(数分に1回程度)でランダムにガラスを叩く。投下位置・叩く位置はランダム(カーソル位置にはこだわらない)。ステータスバーに自動モードのON/OFF状態を表示する
- **設定画面(`,`)**: 効果音・ステータスオーバーレイ・自動モードのON/OFFを一覧表示し、上下キーで選択・Enter/スペースで切替・Escで閉じる(それぞれ`s`/`v`/`a`キーと状態を共有する、分かりやすい入口)
- **効果音**: 気泡・餌やり・投薬・発症・治療・空腹・ガラスを叩く・捕食の瞬間に短いビープ音を鳴らす(`s` でON/OFF、既定はON)。外部音源ファイルは使わず、周波数スイープ+速い減衰の正弦波をその場で合成している
- **時間制御**: `p` で全体を一時停止、`[` `]` で速度を段階変更(空腹度減少・成長・孵化・寿命など時間経過ロジック全体に倍率がかかる)

## 状態の永続化

`~/.config/aquaterm/state.json` に各魚(サメ含む)の種類・成長段階・空腹度・病気状態・年齢・死亡演出の進行度・座標、卵・餌・薬・カニの状態、経過時間を保存します。`q` 終了時に保存し、起動時に読み込んで再開します(ファイルが無ければ初期状態で開始)。血飛沫・血の滲みなど一瞬〜数秒で消える演出、自動モードのON/OFFは保存対象外です(次回起動時は既定状態に戻ります)。

## テスト

```sh
cargo test
```

空腹度の減衰・成長判定(通常成長+成魚後の追加サイズ成長)・繁殖(個体数上限を含む)・寿命/老齢確定産卵・病気の発症/治療・死亡の猶予と死亡演出後の消滅・サメの捕食/追跡速度/成長・逃走(回り込み・逃走コスト)・ランダムダッシュ・血飛沫演出・自動モード・効果音イベントの発火などの育成ロジックを、84件の単体テストで検証しています。オーディオ出力デバイスが無い環境でも `cargo build` / `cargo test` は問題なく通ります(効果音の初期化失敗は握りつぶされます)。
